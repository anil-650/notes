## LAMBDA function

- lambda function is small anonymous
- A function without a function name is anonymous function.
- It can take any number of arguments but can have only one expression.

```py
# Normal Function
def my_func():
    print("Hello world")

my_func()

# Anonymous Function with 'lambda'
x = lambda a:a=a+10
print(x(5))

# Output : 15
```
